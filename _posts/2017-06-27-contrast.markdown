## Contrast

**Note**

Create a `master`-branch if you want to use this theme as root folder on `yourname.github.io`.

**Features**

 - mobile first
 - syntax highlighting
 - [disqus](https://disqus.com/) / [isso](http://posativ.org/isso/) support
 - style customizable
